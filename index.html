설치 안내

"Windows의 PC 보호" 경고가 뜨나요?

새로 나온 앱이라 Windows(SmartScreen)가 아직 익숙하지 않아 경고를 띄우는 것뿐입니다. 바이러스가 아니며 안전합니다. 아래대로 진행하면 정상 설치됩니다.

  1. 파란 경고 창에서 추가 정보(More info)를 클릭합니다.
  2. 아래에 나타나는 실행(Run anyway) 버튼을 누릅니다.
  3. 설치가 정상적으로 진행됩니다. (평판이 쌓이면 경고는 사라집니다.)

💡 다운로드가 차단되면, 브라우저 다운로드 목록에서 ⋯ → 유지(Keep)를 눌러 저장하세요.

FAQ

정말 무료인가요?
네, 무료로 사용할 수 있습니다. 앱에 소량의 광고가 표시되며, 원하시면 앱 안에서 일회성 결제로 광고를 영구 제거할 수 있습니다.
안전한가요? 바이러스 아닌가요?
안전합니다. 처음 설치 시 Windows SmartScreen 경고가 나올 수 있는데, 이는 신생 앱이라 아직 평판이 쌓이지 않아서일 뿐입니다. 위 "설치 안내"대로 진행하시면 됩니다.
자동으로 업데이트되나요?
네. 한 번 설치하면 새 버전이 나올 때 자동으로 내려받아 재시작 시 적용됩니다. 매번 다시 설치할 필요가 없습니다.
Mac도 지원하나요?
현재는 Windows 10/11 전용입니다. macOS 버전은 준비 중입니다.
